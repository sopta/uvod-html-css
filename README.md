# Czechitas 1. blok

**Webová stránka**: Phishing: Nebezpečí digitálního podvodu

## Instalace npm balíčků
```
npm install
```

## Spuštění Cypress testů
```
npm run cy:open
```

## Struktura projektu

/cypress/e2e/ - obsahuje všechny testy

/images/ - obrázky projektu

/pages/ - ostatní stránky pro navigaci

/styles/ - CSS styly
